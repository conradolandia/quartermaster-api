# Security Policy

## Supported Versions

The latest version on the `master` branch is supported.

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it by opening a private security advisory on GitHub:

https://github.com/star-fleet-tours/quartermaster-api/security/advisories/new

Alternatively, you can report it by email. Please try to be as explicit as possible, describing all the steps and example code to reproduce the issue.

Do not open a public issue for security vulnerabilities.

## Public Discussions

Please do not publicly discuss a potential security vulnerability. Report it privately first so a fix can be developed before disclosure.
